# CoFoundr AI — Phase 1

The Phase 1 frontend foundation for CoFoundr AI: a premium landing experience for an AI-powered founding team.

## Included

- Responsive dark-mode landing page with animated hero, agent cards, AI CEO showcase, workflow, benefits, and footer
- Reusable component architecture using Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, Lucide, and shadcn-style UI primitives
- Prepared folders for shared components, hooks, types, styles, and utilities
- Placeholder routes for `/dashboard`, `/workspace`, `/chat`, `/investor`, and `/settings`

## Intentionally not included

AI agents, authentication, database storage, APIs, backend logic, and product workflow behavior are reserved for future phases.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Verify production build

```bash
npm run build
```
